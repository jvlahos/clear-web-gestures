A web experiment by Jeff Vlahos, UI/UX designer and front-end developer, and designer at Upstatement.

Clear Web Gestures

Clear Web Gestures is an experimental effort to remind the makers of the web that gestural experiences aren’t only limited to native apps. And to prove it, I recreated some of the gestures from one of the native land’s most infamous gestural applications, Clear.1

So go ahead. Touch it.
